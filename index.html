My React Widget 배포 페이지

이 위젯을 사용하려면 아래와 같은 코드를 사용하세요:


<script src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>
<script src="https://ronanam.github.io/create-widget/create-widget.umd.js"></script>

<div id="widget-container"></div>

<script>
  window.MyWidget.mount(document.getElementById("widget-container"));
</script>
